# Blog App

Welcome to the Blog App! This is a full-stack application built with MongoDB, Express, React, and Node.js (MERN stack).

## Installation
cd blog-app
npm install
cd ../server
npm install

### Prerequisites
- Node.js
- npm
- MongoDB
#### Setup 
- Crete a .env file on /server  with given values

- PORT=5000
- MONGO_URL=mongodb://localhost:27017/blog_app
